# Security Policy

This repository defines license and governance text. Report issues if a change:

- weakens misuse boundaries;
- authorizes attack, theft, fraud, bypass, unsafe surveillance, or criminal
  automation;
- removes no-warranty or no-liability language;
- falsely claims SPDX, OSI, provider, school, or legal endorsement;
- exposes private data, credentials, or unpublished source bodies.

SEL/SECL are educational governance tools and are not legal advice. Projects
using them must still perform their own legal, safety, and security review.
